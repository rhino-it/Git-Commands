Git Commands

  • mkdir - create folder
  • vi - create file
  • esc :wq - save and quit
  • ls - check the status(condition)
  • git init - create rep
  • git config --global user.name "" - register user
  • git config --global user.email "" - register email
  • git branch - check countity of branches
  • git branch ******* - create a new branch
  • git checkout - switch to another branch
  • rm -rf .git - delete all what you did
  • $ git push -u origin ****** - send to another branch
